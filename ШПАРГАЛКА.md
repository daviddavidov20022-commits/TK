# 🚛 ШПАРГАЛКА — Калькулятор ТК

## Что нужно сделать:

### Шаг 1: Закрыть программу (редактор)
Закрой VS Code / редактор полностью.

### Шаг 2: Переименовать папку
Переименуй папку вручную в проводнике:
- БЫЛО: `D:\KAL'KYLAYTOR TK`  
- СТАЛО: `D:\kalTK`

### Шаг 3: Открыть проект заново
Открой переименованную папку `D:\kalTK` в редакторе.

### Шаг 4: Написать агенту
Напиши агенту:
```
/setup
```
Агент прочитает workflow и автоматически:
1. Обновит package.json
2. Установит зависимости (npm install)
3. Запустит сервер

### Шаг 5: Настроить API ключи
Открой файл `.env` и впиши свои ключи:
- `DELLIN_APP_KEY` — ключ от Деловых Линий (получить на https://dev.dellin.ru)
- `DELLIN_LOGIN` / `DELLIN_PASSWORD` — логин/пароль от ДЛ
- ПЭК работает без ключа через публичный API

### Шаг 6: Открыть калькулятор
Перейди в браузер: http://localhost:3000

---

## Как пользоваться:
1. **База товаров** — добавь товары (артикул, название, габариты, вес)
2. **Калькулятор** — введи артикул товара → введи город/адрес доставки → нажми "Рассчитать"
3. Увидишь цены от ДЛ и ПЭК по всем вариантам (терминал→терминал, адрес→адрес и т.д.)
